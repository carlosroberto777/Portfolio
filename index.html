Tecnologias

HTML

HTML (HyperText Markup Language) é a linguagem padrão usada para criar páginas na web. Ele estrutura o conteúdo de um site, definindo elementos como títulos, parágrafos, imagens, links, listas, botões, entre outros. Em outras palavras, o HTML é como o "esqueleto" de uma página web — ele organiza e descreve o que será exibido no navegador.

CSS

CSS (Cascading Style Sheets) é a linguagem usada para estilizar páginas web criadas com HTML. Ele define cores, fontes, tamanhos, espaçamentos, posicionamento e até animações dos elementos da página. Enquanto o HTML estrutura o conteúdo, o CSS define como esse conteúdo vai aparecer visualmente no navegador.

GIT

Git é um sistema de controle de versão usado principalmente no desenvolvimento de software. Ele permite que você salve o histórico do seu código, acompanhe mudanças, trabalhe em equipe e volte no tempo caso algo dê errado. Em outras palavras, o Git funciona como uma “máquina do tempo” para seus projetos de programação.

JAVASCRIPT

JavaScript (JS) é uma linguagem de programação usada para tornar páginas web interativas e dinâmicas. Com ele, você pode criar ações, animações, validações de formulários, mostrar ou esconder elementos e muito mais. Enquanto o HTML estrutura a página e o CSS estiliza, o JavaScript dá vida e funcionalidade a ela.

BOOTSTRAP

Bootstrap é um framework front-end que facilita a criação de sites responsivos, bonitos e funcionais de forma rápida. Ele oferece um conjunto de códigos prontos em HTML, CSS e JavaScript, como botões, menus, formulários, grids e muito mais. Em resumo: o Bootstrap ajuda você a criar interfaces modernas sem ter que escrever tudo do zero.
